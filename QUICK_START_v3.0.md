# Master Improviser Mode - Quick Start Guide

## What You Just Got üéπüé∏

Your Melody Generator's **Pianist/Guitarist mode** is now a **Master Improviser** with **39 sophisticated improvisational techniques** that make it sound like a virtuoso player!

## How to Use It

### Step 1: Load the Script
1. Open REAPER
2. Actions > Show action list > ReaScript > Load
3. Select: `jtp gen_GenMusic_Melody Generator Dialog.lua`

### Step 2: Select Pianist/Guitarist Mode
1. Choose "Manual" when prompted
2. Set up your root note and scale (or use auto-detect from regions)
3. Set **Number of Voices** to 3-5 (for pianist/guitarist feel)
4. When asked for **Polyphony Mode**, choose: **"Pianist/Guitarist (Chords + Fills)"**

### Step 3: Enjoy Master-Level Improvisation!
The script will now generate sophisticated improvisations with:
- **Chords** on strong beats with smooth voice leading
- **Fills** between chords using 39 different techniques
- **Motif development** where musical ideas return transformed
- **Sophisticated ornaments** like trills, turns, grace notes
- **Polyrhythms** including triplets and quintuplets
- **Register exploration** with dramatic leaps and cascades
- **Harmonic sophistication** with extensions and alterations

## What to Listen For üëÇ

### Ornamentation
- Quick **trills** on sustained notes
- Elegant **turns** at phrase endings
- **Grace note clusters** leading into chords
- **Chromatic approaches** (jazz-style outside notes)

### Rhythmic Interest
- **Triplet runs** flowing between chords
- **Quintuplet flourishes** for complexity
- **Syncopated riffs** with off-beat emphasis
- **Rubato passages** that speed up and slow down naturally

### Dramatic Moments
- **Wide interval jumps** (octave leaps)
- **Cascade descents** rushing downward
- **Ascending rockets** shooting up to high register
- **Octave-displaced arpeggios**

### Musical Intelligence
- **Motifs that return** - you'll hear musical ideas come back transformed:
  - Higher or lower (sequence)
  - Backwards (retrograde)
  - Upside down (inversion)
  - Fragmented into pieces
  - With different rhythm

### Harmonic Color
- **Extended chords** (9ths, 11ths, 13ths)
- **Altered tones** (chromatic substitutions)
- **Tension and resolution** arcs
- **Modal flavors** (characteristic scale colors)

## Key Parameters

### Number of Voices
- **3:** Guitar-style (thinner chords, more room for fills)
- **4-5:** Piano-style (full chords, balanced)
- **6+:** Orchestral (dense harmony)

### Theory Weight
- **0.0-0.3:** Wild, creative, outside playing
- **0.4-0.6:** Balanced (recommended) ‚≠ê

## Troubleshooting

### "It sounds too busy"
- Reduce Number of Voices to 3
- Increase Measures to 4 or more (gives more space)
- The 30% silence probability will help with breathing

### "I want more outside/jazz playing"
- Lower Theory Weight to 0.2-0.4
- Use scales like Dorian, Lydian, Altered
- Listen for chromatic approaches and altered fills

### "I want smoother, more classical sound"
- Increase Theory Weight to 0.7-0.9
- Use Major, Minor, or modal scales
- 4 voices for full piano sound
- Listen for trills, turns, and smooth arpeggios

### "How do I hear specific techniques?"
Run it multiple times! With 39 different fill types selected randomly, you'll hear different techniques each time. The more you generate, the more variety you'll experience.
